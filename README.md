# PostureCop

Setup:
1. Install Python
2. Create a python virtual environment: python -m venv <environment name>
3. Activate the virtual environment
			i. cd into <environmentname>
			ii. cd into Scripts folder
			iii. run 'activate'
      iV. cd back up to the project folder
4. Install Ultralytics: pip install ultralytics
5. Install Pytorch: 		pip install torch torchvision torchaudio
      i. 		For a version that's known to work with the Lenovo P16v: pip install torch==2.2.0 torchvision==0.17.0 torchaudio==2.2.0 --index-url https://download.pytorch.org/whl/cu118
6. run the app: python fhp_detector.py
